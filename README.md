# Joomshopping Payment Payme gateway

## Требования

+ [Регистрация](https://merchant.paycom.uz/#/register) в кабинете поставщика услуг Paycom™
+ [Joomla](https://www.joomla.org/) 3.x и выше
+ [JoomShopping](https://joomshopping.pro/) 4.15 и выше
